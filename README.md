
Notepad Pro is a powerful, feature-rich, and privacy-focused web-based text editor. Designed as a progressive web application (PWA) alternative to traditional desktop notepads, it offers a seamless writing experience with modern UI/UX elements, rich text formatting, and robust file handling capabilities—all within a single HTML file.

Built by Pratama Digital, this tool requires no backend installation. Simply open the HTML file in any modern browser to start writing.


✨ Features
📝 Rich Text Editing
Formatting: Bold, Italic, Underline, Strikethrough.
Typography: Wide selection of fonts (Inter, JetBrains Mono, Roboto, etc.) and adjustable font sizes.
Colors: Font color picker and text highlighter (background color).
Layout: Text alignment (Left, Center, Right) and bullet lists.
Links: Insert and remove hyperlinks easily.
Code View: Toggle between the Visual Editor and HTML Source Mode to view or edit the underlying code directly.
📂 File Management
Multi-Tab Interface: Open multiple files in separate tabs within a single window.
Import: Open files directly from your computer. Supports .txt, .html, .rtf, .docx, and .pdf.
Export: Save your work locally or download as Text, RTF, PDF, or HTML.
Auto-Save: Automatically saves your session to the browser's LocalStorage.
🎨 User Interface
Dark/Light Mode: Toggle between themes.
Responsive Design: Works on desktops, tablets, and mobile devices.
Modern UI: Clean Indigo/Purple theme design.
🚀 Getting Started
Installation
No installation is required.

Download the index.html file.
Open it with any modern web browser (Chrome, Firefox, Edge, Safari).
⌨️ Keyboard Shortcuts
Shortcut	Action
Ctrl + S	Save to LocalStorage
Ctrl + Shift + S	Save As (Export Menu)
Ctrl + N	New Tab
Ctrl + B	Bold
Ctrl + K	Insert Link
Ctrl + H	Find & Replace
